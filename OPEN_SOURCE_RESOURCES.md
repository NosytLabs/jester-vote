# Open Source Resources for Lolcow/Streamer Documentation

## Stream Archiving & Documentation Tools

### Chat Loggers
- **twitch-chat-logger** (bernardopires) - PostgreSQL-based Twitch chat logger
  - https://github.com/bernardopires/twitch-chat-logger
  - 64 stars, MIT License

- **Twitch_Batch_Logger** (gamedeff) - Batch logger for multiple channels
  - https://github.com/gamedeff/Twitch_Batch_Logger
  - 11 stars, MIT License

- **spaghetti-logger** (chfoo) - Twitch.tv chat logger
  - https://github.com/chfoo/spaghetti-logger
  - 9 stars

- **twitchlogger** (mlvzk) - Minimal resource usage logger
  - https://github.com/mlvzk/twitchlogger
  - 9 stars, MIT License

### Video Archiving
- **youtube-dl** - Download videos from YouTube and other sites
  - https://github.com/ytdl-org/youtube-dl
  - The standard for video archiving

- **yt-dlp** - Active fork of youtube-dl with more features
  - https://github.com/yt-dlp/yt-dlp
  - Better support for livestreams and clips

- **streamlink** - Extract streams from various services
  - https://github.com/streamlink/streamlink
  - CLI utility for extracting video streams

### Documentation & Wiki Tools
- **MediaWiki** - The software behind Wikipedia
  - https://github.com/wikimedia/mediawiki
  - Good for creating lolcow documentation wikis

- **Outline** - Open source wiki and knowledge base
  - https://github.com/outline/outline
  - Modern, beautiful documentation platform

- **Wiki.js** - Open source wiki platform
  - https://github.com/Requarks/wiki
  - Feature-rich, modern wiki software

### Community Platforms
- **Discourse** - Open source discussion platform
  - https://github.com/discourse/discourse
  - Good for lolcow discussion forums

- **Flarum** - Simple, fast forum software
  - https://github.com/flarum/core
  - Modern, lightweight forum

- **Lemmy** - Reddit alternative (federated)
  - https://github.com/LemmyNet/lemmy
  - Good for creating community instances

### Image/Video Hosting
- **Piwigo** - Open source photo gallery
  - https://github.com/Piwigo/Piwigo
  - Good for archiving screenshots

- **Chevereto** - Image hosting software
  - https://github.com/chevereto/chevereto
  - Self-hosted image hosting

### Data Collection
- **waybackpy** - Python interface to Wayback Machine
  - https://github.com/akamhy/waybackpy
  - Archive web pages automatically

- **archivebox** - Self-hosted web archiving
  - https://github.com/ArchiveBox/ArchiveBox
  - Archive websites locally

## APIs for Streamer Data

### Twitch API
- **Twitch API v2** - Official Twitch API
  - https://dev.twitch.tv/docs/api/
  - Get streamer info, clips, chat data

### YouTube API
- **YouTube Data API v3**
  - https://developers.google.com/youtube/v3
  - Get video data, comments, channel info

### Kick API
- **Kick.com API** (unofficial)
  - Various community APIs available
